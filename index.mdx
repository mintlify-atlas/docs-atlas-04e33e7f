Bridgex is an open-source desktop application built in Rust that gives you a graphical interface for converting common document formats into Markdown. It wraps the [Markitdown-rs](https://github.com/uhobnil/markitdown-rs) library in a clean split-pane window — file input on the left, live Markdown preview on the right — so you can convert, edit, and save without touching the command line.

## What Bridgex Does

Bridgex converts documents into clean Markdown text using the Markitdown-rs library, then opens that text directly in a built-in code editor. You can review and refine the output, then save a `.md` file wherever you like — no configuration required for basic use.

## Get Started in Three Steps

<Steps>
  <Step title="Install Bridgex">
    Choose your platform — install via Cargo, pip, or download a prebuilt binary. See the [Installation guide](/installation) for all options.
  </Step>
  <Step title="Open a File">
    Launch the app and press **Ctrl+O** (or use **File → Open**) to pick a supported file. Bridgex converts it to Markdown instantly.
  </Step>
  <Step title="Edit and Save">
    Review the output in the editor pane, make any changes, then press **Ctrl+S** (or **File → Save**) to write the `.md` file to disk.
  </Step>
</Steps>

<Note>
  On Linux, install from [SourceForge](https://sourceforge.net/projects/bridgex/) — the `pip install bridgex` wheel is only available for Windows and macOS.
</Note>
