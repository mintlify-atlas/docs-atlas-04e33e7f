Bridgex is an open-source desktop application (v0.2.1) built in Rust with the [Freya](https://freyaui.dev/) UI framework. It wraps the [Markitdown](https://github.com/microsoft/markitdown) crate to convert common document formats into Markdown, then opens the result in a built-in editor so you can review, refine, and save — no command line required.

## What Bridgex Does

Bridgex converts documents into clean Markdown text using the Markitdown crate, then opens that text in a built-in code editor. You can review and refine the output, then save a `.md` file wherever you like — no configuration required for basic use.

## Get Started in Three Steps

<Steps>
  <Step title="Install Bridgex">
    Choose your platform — install via Cargo, pip, or download a prebuilt binary. See the [Installation guide](/installation) for all options.
  </Step>
  <Step title="Open a File">
    Launch the app and press **Ctrl+O** (or use **File → Open**) to pick a supported file. Bridgex converts it to Markdown instantly.
  </Step>
  <Step title="Edit and Save">
    Review the output in the editor, make any changes, then press **Ctrl+S** (or **File → Save**) to write the `.md` file to disk.
  </Step>
</Steps>

<Note>
  On Linux, install from [SourceForge](https://sourceforge.net/projects/bridgex/) — the `pip install bridgex` wheel is only available for Windows and macOS.
</Note>
